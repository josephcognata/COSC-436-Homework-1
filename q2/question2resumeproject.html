Project: Cutest Paw

Description

This project was a pet focused social media website where you could post images and talk about your pets. Create profile for you and your pets and upload an avatar, images, status updates, bio and likes and dislikes. It also gave you the ability to add friends on the website and comment on their activities. For the project we use the MERN stack which included MongoDB for the database, Express and Node JS for the backend, and React JS for the front end. The website is hosted at cutestpaw.herokuapp.com

                ReactSession.setStoreType('localStorage');
